# Lesson 2: HTML Forms

A form is used to collect data from users.

Examples:

* Login page
* Registration page
* Contact form
* Search box
* Feedback form

---

# What is a Form?

HTML provides the `<form>` tag.

Example:

```html
<form>
</form>
```

Everything related to user input goes inside the form.

---

# Text Input

The most common input field.

```html
<input type="text">
```

Output:

A textbox where users can type.

Example:

```html
<input type="text" placeholder="Enter your name">
```

### Placeholder

```html
placeholder="Enter your name"
```

Displays a hint inside the textbox.

---

# Labels

Labels describe inputs.

```html
<label>Name</label>
<input type="text">
```

Better example:

```html
<label>Full Name</label>
<br>
<input type="text">
```

Output:

```text
Full Name
[____________]
```

---

# Password Field

Used for passwords.

```html
<input type="password">
```

Example:

```html
<label>Password</label>
<input type="password">
```

Characters appear as:

```text
*******
```

instead of visible text.

---

# Email Input

```html
<input type="email">
```

Example:

```html
<label>Email</label>
<input type="email">
```

Browsers can perform basic email validation.

---

# Number Input

```html
<input type="number">
```

Example:

```html
<label>Age</label>
<input type="number">
```

Only numeric values are expected.

---

# Buttons

```html
<button>Submit</button>
```

Example:

```html
<button>Register</button>
```

Output:

```text
[ Register ]
```

---

# Submit Button

Used to submit a form.

```html
<button type="submit">
    Submit
</button>
```

or

```html
<input type="submit">
```

---

# Checkbox

Used when multiple selections are allowed.

Example:

```html
<input type="checkbox">
```

Real example:

```html
<label>
    <input type="checkbox">
    I agree to Terms
</label>
```

Output:

```text
☐ I agree to Terms
```

User can check or uncheck it.

---

# Radio Buttons

Used when only one option can be selected.

Example:

```html
<label>
    <input type="radio" name="gender">
    Male
</label>

<label>
    <input type="radio" name="gender">
    Female
</label>
```

Output:

```text
○ Male
○ Female
```

Because both share:

```html
name="gender"
```

only one can be selected.

---

# Dropdown (Select)

```html
<select>
    <option>India</option>
    <option>USA</option>
    <option>Canada</option>
</select>
```

Output:

```text
[ India ▼ ]
```

User can choose one option.

---

# Text Area

For long text.

Example:

```html
<textarea></textarea>
```

Real example:

```html
<label>Message</label>

<textarea rows="5" cols="30"></textarea>
```

Output:

Large typing area.

Useful for:

* Comments
* Feedback
* Reviews

---

# Required Field

Makes input mandatory.

```html
<input type="text" required>
```

Example:

```html
<label>Name</label>
<input type="text" required>
```

Browser won't submit if empty.

---

# Min and Max Length

```html
<input
  type="text"
  minlength="3"
  maxlength="20">
```

Rules:

* Minimum 3 characters
* Maximum 20 characters

---

# Complete Registration Form Example

```html
<!DOCTYPE html>
<html>
<head>

<body>

<h1>User Registration</h1>

<form>

    <br><br>

    <label>Email</label>
    <br>
    <input
      type="email"
      placeholder="Enter Email">

    <br><br>

    <label>Password</label>
    <br>
    <input type="password">

    <br><br>

    <label>Age</label>
    <br>
    <input type="number">

    <br><br>

    <label>Gender</label>

    <br>

    <input
      type="radio"
      name="gender">
    Male

    <input
      type="radio"
      name="gender">
    Female

    <br><br>

    <label>Country</label>

    <br>

    <select>

        <option>USA</option>

    <br><br>

    <label>
      <input type="checkbox">

    <br><br>

    <button type="submit">
        Register
    </button>

</form>

</body>
</html>
```

---

# Why This Matters for Angular

Later in Angular you'll see code like:

```html
<input [(ngModel)]="username">
```

or

```html
<input formControlName="email">
```

These Angular features work on top of the HTML form elements you're learning now.

So before Angular forms, you must be comfortable with:

✅ `<form>`
✅ `<input>`
✅ `<label>`
✅ `<button>`
✅ `<select>`
✅ `<textarea>`
✅ `<checkbox>`
✅ `<radio>`
✅ Validation attributes (`required`, `minlength`, etc.)
