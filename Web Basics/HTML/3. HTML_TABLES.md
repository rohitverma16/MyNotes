# Lesson 3: HTML Tables

Tables display data in rows and columns.

Example:

| Name | Age | City   |
| ---- | --- | ------ |
| John | 25  | Delhi  |
| Mike | 30  | Mumbai |

---

# Basic Table Structure

```html
<table>
</table>
```

Inside a table we use:

```html
<tr>   Table Row
<th>   Table Header
<td>   Table Data
```

---

# Simple Table Example

```html
<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>

    <tr>
        <td>John</td>
        <td>25</td>
    </tr>

    <tr>
        <td>Mike</td>
        <td>30</td>
    </tr>
</table>
```

Output:

| Name | Age |
| ---- | --- |
| John | 25  |
| Mike | 30  |

---

# Understanding Each Tag

## Table Row

```html
<tr>
</tr>
```

Creates a row.

Example:

```html
<tr>
   <td>John</td>
   <td>25</td>
</tr>
```

---

## Table Header

```html
<th>Name</th>
```

Headers are bold by default.

Example:

```html
<tr>
   <th>Name</th>
   <th>Age</th>
</tr>
```

---

## Table Data

```html
<td>John</td>
```

Regular cell data.

---

# Employee Table Example

```html
<table border="1">

<tr>
   <th>ID</th>
   <th>Name</th>
   <th>Department</th>
</tr>

<tr>
   <td>101</td>
   <td>Rahul</td>
   <td>IT</td>
</tr>

<tr>
   <td>102</td>
   <td>Aman</td>
   <td>HR</td>
</tr>

<tr>
   <td>103</td>
   <td>Priya</td>
   <td>Finance</td>
</tr>

</table>
```

Output:

```text
+-----+-------+-----------+
| ID  | Name  | Department|
+-----+-------+-----------+
|101  |Rahul  | IT        |
|102  |Aman   | HR        |
|103  |Priya  | Finance   |
+-----+-------+-----------+
```

---

# Column Span

Sometimes one cell should cover multiple columns.

```html
<td colspan="2">
    Employee Details
</td>
```

Example:

```html
<table border="1">

<tr>
    <td colspan="2">
        Employee Information
    </td>
</tr>

<tr>
    <td>Name</td>
    <td>John</td>
</tr>

</table>
```

---

# Row Span

One cell covers multiple rows.

```html
<td rowspan="2">
    IT
</td>
```

Example:

```html
<table border="1">

<tr>
    <td rowspan="2">IT</td>
    <td>John</td>
</tr>

<tr>
    <td>Mike</td>
</tr>

</table>
```

---

# Table Sections

Professional tables use:

```html
<thead>
<tbody>
<tfoot>
```

Example:

```html
<table border="1">

<thead>
<tr>
    <th>Name</th>
    <th>Salary</th>
</tr>
</thead>

<tbody>
<tr>
    <td>John</td>
    <td>50000</td>
</tr>

<tr>
    <td>Mike</td>
    <td>60000</td>
</tr>
</tbody>

<tfoot>
<tr>
    <td>Total</td>
    <td>110000</td>
</tr>
</tfoot>

</table>
```

---

# Why Tables Matter in Angular

Later you'll generate rows dynamically.

Example Angular template:

```html
<tr *ngFor="let employee of employees">
   <td>{{ employee.id }}</td>
   <td>{{ employee.name }}</td>
   <td>{{ employee.department }}</td>
</tr>
```

This is why understanding HTML tables first is important.

# Semantic HTML

Now let's learn one of the most important HTML concepts.

## What is Semantic HTML?

Semantic tags describe the meaning of content.

Instead of:

```html
<div>
</div>
```

Use meaningful tags:

```html
<header>
<nav>
<main>
<section>
<article>
<footer>
```

This improves:

* Readability
* Accessibility
* SEO
* Maintainability

---

# Header

Top section of a webpage.

```html
<header>
    <h1>My Website</h1>
</header>
```

Usually contains:

* Logo
* Title
* Navigation

---

# Navigation

```html
<nav>
    <a href="#">Home</a>
    <a href="#">About</a>
    <a href="#">Contact</a>
</nav>
```

Contains menus and links.

---

# Main

Main content area.

```html
<main>
    Content Here
</main>
```

Each page should generally have one `<main>`.

---

# Section

Groups related content.

```html
<section>
    <h2>Skills</h2>
    <p>Angular Development</p>
</section>
```

---

# Article

Independent content.

Example:

```html
<article>
    <h2>Angular Basics</h2>
    <p>Angular is a frontend framework.</p>
</article>
```

Useful for:

* Blog posts
* News articles
* Tutorials

---

# Aside

Side content.

```html
<aside>
    Related Articles
</aside>
```

Usually appears as a sidebar.

---

# Footer

Bottom section of a page.

```html
<footer>
    Copyright 2026
</footer>
```

Contains:

* Copyright
* Contact info
* Social links

---

# Complete Semantic Page Example

```html
<!DOCTYPE html>
<html>

<head>
    <title>My Blog</title>
</head>

<body>

<header>
    <h1>Tech Blog</h1>
</header>

<nav>
    <a href="#">Home</a>
    <a href="#">Courses</a>
    <a href="#">Contact</a>
</nav>

<main>

<section>
    <h2>Latest Tutorials</h2>

    <article>
        <h3>Angular Introduction</h3>
        <p>Learn Angular from scratch.</p>
    </article>

    <article>
        <h3>JavaScript Basics</h3>
        <p>Learn JavaScript fundamentals.</p>
    </article>

</section>

</main>

<footer>
    Copyright 2026
</footer>

</body>

</html>
```
