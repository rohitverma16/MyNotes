# Lesson 1: What is HTML?

**HTML (HyperText Markup Language)** is the standard language used to create web pages.

HTML describes:

* The structure of a webpage
* Headings
* Paragraphs
* Images
* Links
* Forms
* Tables

Think of HTML as the **skeleton** of a website.

Example:

```html
<h1>My First Website</h1>
<p>Welcome to my website.</p>
```

Output:

```
My First Website
Welcome to my website.
```

---

# Basic Structure of an HTML Page

Every HTML page follows this structure:

```html
<!DOCTYPE html>
<html>
<head>
    <title>My Website</title>
</head>

<body>
    <h1>Hello World</h1>
</body>
</html>
```

Let's understand each part.

## 1. <!DOCTYPE html>

```html
<!DOCTYPE html>
```

Tells the browser:

> "This document uses HTML5."

It should always be the first line.

---

## 2. html Tag

```html
<html>
</html>
```

The root element of the webpage.

Everything goes inside this tag.

---

## 3. head Tag

```html
<head>
</head>
```

Contains information about the page.

Examples:

* Page title
* CSS files
* Meta tags
* Scripts

---

## 4. title Tag

```html
<title>My Website</title>
```

Appears in the browser tab.

Example:

```
Chrome Tab:
My Website
```

---

## 5. body Tag

```html
<body>
</body>
```

Everything visible on the page goes here.

Examples:

* Text
* Images
* Buttons
* Forms

---

# HTML Elements

HTML is made of elements.

Example:

```html
<h1>Hello</h1>
```

Breakdown:

```html
<h1>       Opening Tag
Hello      Content
</h1>      Closing Tag
```

This complete structure is called an **element**.

---

# Headings

HTML provides six heading levels.

```html
<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Section Heading</h3>
<h4>Small Heading</h4>
<h5>Smaller Heading</h5>
<h6>Smallest Heading</h6>
```

Output hierarchy:

```
H1 (largest)
H2
H3
H4
H5
H6 (smallest)
```

Use only one major `<h1>` per page in most cases.

---

# Paragraphs

Paragraph tag:

```html
<p>This is a paragraph.</p>
```

Example:

```html
<p>I am learning Angular.</p>

<p>HTML is the first step.</p>
```

Output:

```
I am learning Angular.

HTML is the first step.
```

---

# Line Break

To move text to a new line:

```html
Hello<br>
World
```

Output:

```
Hello
World
```

---

# Horizontal Line

```html
<hr>
```

Output:

A horizontal separator line.

Example:

```html
<h1>Profile</h1>

<hr>

<p>Information here</p>
```

---

# Comments

Comments are ignored by browsers.

```html
<!-- This is a comment -->
```

Useful for notes.

Example:

```html
<!-- Header Section -->

<h1>My Website</h1>
```

---

# HTML Attributes

Attributes provide additional information.

Example:

```html
<a href="https://google.com">
  Google
</a>
```

Here:

```html
href="https://google.com"
```

is an attribute.

General format:

```html
<tag attribute="value">
```

---

# Links

Anchor tag:

```html
<a href="https://google.com">
    Open Google
</a>
```

When clicked:

* Opens Google

Another example:

```html
<a href="https://github.com">
    GitHub
</a>
```

---

# Images

Display images using:

```html
<img src="image.jpg">
```

Better example:

```html
<img
    src="cat.jpg"
    alt="Cat Image"
>
```

### Important Attributes

#### src

Image location.

```html
src="cat.jpg"
```

#### alt

Alternative text if image fails.

```html
alt="Cat Image"
```

---

# Lists

Two types:

## Ordered List

Numbered list.

```html
<ol>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ol>
```

Output:

```
1. HTML
2. CSS
3. JavaScript
```

---

## Unordered List

Bullet list.

```html
<ul>
    <li>Apple</li>
    <li>Mango</li>
    <li>Orange</li>
</ul>
```

Output:

```
• Apple
• Mango
• Orange
```

---

# Div Element

Very important.

```html
<div>
    Content Here
</div>
```

`div` is a container.

Example:

```html
<div>
    <h1>Profile</h1>
    <p>Developer</p>
</div>
```

In Angular, you'll use `<div>` constantly.

---

# Span Element

Inline container.

```html
<p>
  My name is
  <span>Rahul</span>
</p>
```

Usually used for styling small pieces of text.
