# Lesson 5: CSS Basics (Very Important for Angular)

## What is CSS?

**CSS (Cascading Style Sheets)** is used to style HTML.

It controls:

* Colors
* Layout
* Spacing
* Fonts
* Animations
* Responsive design

👉 If HTML is the skeleton, CSS is the **skin + design**.

---

# 1. How to Add CSS

There are 3 ways:

---

## 1. Inline CSS (Not recommended)

```html id="css1"
<p style="color:red;">Hello</p>
```

---

## 2. Internal CSS

```html id="css2"
<head>
<style>
p {
  color: blue;
}
</style>
</head>
```

---

## 3. External CSS (BEST — used in Angular)

```html id="css3"
<link rel="stylesheet" href="styles.css">
```

In Angular, each component has its own CSS file.

---

# 2. CSS Syntax

```css id="css4"
selector {
  property: value;
}
```

Example:

```css id="css5"
p {
  color: red;
}
```

---

# 3. Selectors

## Element Selector

```css id="css6"
p {
  color: blue;
}
```

Targets all `<p>` tags.

---

## Class Selector (VERY IMPORTANT)

```css id="css7"
.box {
  color: green;
}
```

HTML:

```html id="css8"
<div class="box">Hello</div>
```

---

## ID Selector

```css id="css9"
#title {
  color: red;
}
```

HTML:

```html id="css10"
<h1 id="title">Heading</h1>
```

---

## Universal Selector

```css id="css11"
* {
  margin: 0;
}
```

Applies to everything.

---

# 4. Colors in CSS

---

## Hex Colors

```css id="css13"
color: #ff0000;
```

---

## RGB

```css id="css14"
color: rgb(255, 0, 0);
```

# 8. Box Model (MOST IMPORTANT CONCEPT)

Every HTML element is a box:

```
+----------------------+
|      Margin          |
|  +----------------+  |
|  |   Border       |  |
|  |  +----------+  |  |
|  |  | Padding  |  |  |
|  |  | Content  |  |  |
|  |  +----------+  |  |
|  +----------------+  |
+----------------------+
```

---

## Example

```css id="css18"
div {
  margin: 20px;
  padding: 15px;
  border: 2px solid black;
}
```

---

# 9. Margin vs Padding

## Margin (outside space)

```css id="css19"
margin: 20px;
```

## Padding (inside space)

```css id="css20"
padding: 20px;
```

---

# 10. Borders

```css id="css21"
div {
  border: 2px solid red;
}
```

* solid
* dashed
* dotted

---

# 11. Width and Height

```css id="css22"
div {
  width: 300px;
  height: 200px;
}
```

---

# 12. Display Property

## Block Element

Takes full width:

```css id="css23"
div {
  display: block;
}
```

---

## Inline Element

Takes only needed space:

```css id="css24"
span {
  display: inline;
}
```

---

## Inline-Block

Best of both:

```css id="css25"
div {
  display: inline-block;
}
```

---

# 13. Flexbox (VERY IMPORTANT for Angular UI)

Used for layout alignment.

---

## Basic Flexbox

```css id="css26"
.container {
  display: flex;
}
```

---

## Center Alignment

```css id="css27"
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

---

## Space Between

```css id="css28"
.container {
  display: flex;
  justify-content: space-between;
}
```

---

# 14. Grid Layout (Advanced Layout System)

```css id="css29"
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
```

Creates 3 equal columns.

---

# 15. Hover Effect

```css id="css30"
button:hover {

  color: white;
}
```

---

# 16. Simple Styled Page Example

```html id="css31"
<!DOCTYPE html>
<html>

<head>
<style>

body {
  font-family: Arial;
  background-color: #f2f2f2;
}

.card {
  width: 300px;
  padding: 20px;
  margin: 50px auto;
  border: 1px solid #ccc;
  border-radius: 10px;
  background-color: white;
  text-align: center;
}

button {
  padding: 10px 15px;
  background-color: green;
  color: white;
  border: none;
}

button:hover {
  background-color: darkgreen;
}

</style>
</head>

<body>

<div class="card">
  <h2>John Doe</h2>
  <p>Angular Developer</p>
  <button>Follow</button>
</div>

</body>

</html>
```

# Why CSS is Important for Angular

In Angular:

* Each component has its own CSS file
* Layouts use Flexbox/Grid
* UI is built using reusable styles
* Dynamic classes are applied like:

```html id="ngcss"
<div [ngClass]="{ active: isActive }"></div>
```
