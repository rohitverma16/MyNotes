# 1. Images (Advanced Usage)

We already saw `<img>`, now let’s go deeper.

Basic:

```html id="img1"
<img src="cat.jpg" alt="Cat Image">
```

---

## Image Size Control

```html id="img2"
<img src="cat.jpg" alt="Cat" width="300" height="200">
```

* `width` → horizontal size
* `height` → vertical size

---

## Image from URL

```html id="img3"
<img src="https://example.com/image.jpg" alt="Online Image">
```

Used in real apps (profiles, products, dashboards).

---

## Why ALT is Important

```html id="img4"
<img src="dog.jpg" alt="Golden Retriever dog sitting">
```

ALT is used for:

* SEO (search engines)
* Accessibility (screen readers)
* Fallback if image fails

---

# 2. Audio in HTML

Used for music, podcasts, voice notes.

```html id="audio1"
<audio controls>
    <source src="song.mp3" type="audio/mpeg">
</audio>
```

---

## Key Attributes

### controls

Adds play/pause buttons:

```html id="audio2"
<audio controls>
```

---

## Multiple Formats (Best Practice)

```html id="audio3"
<audio controls>
    <source src="song.mp3" type="audio/mpeg">
    <source src="song.ogg" type="audio/ogg">
</audio>
```

Browser picks supported format.

---

# 3. Video in HTML

Used for tutorials, reels, demos.

```html id="video1"
<video width="400" controls>
    <source src="video.mp4" type="video/mp4">
</video>
```

---

## Autoplay Video (use carefully)

```html id="video2"
<video autoplay muted controls>
```

⚠️ Most browsers block autoplay unless muted.

---

## Loop Video

```html id="video3"
<video controls loop>
```

Used in contact pages.

# 5. HTML Entities

Entities are used for special characters.

---

## Common Entities

| Symbol | Entity   |
| ------ | -------- |
| <      | `&lt;`   |
| >      | `&gt;`   |
| &      | `&amp;`  |
| "      | `&quot;` |
| '      | `&apos;` |

---

## Example

```html id="entity1"
<p>5 &lt; 10</p>
```

Output:

```
5 < 10
```

---

## More Examples

```html id="entity2"
<p>AT&amp;T Company</p>
<p>&quot;Angular&quot; Framework</p>
```

---

## Why Entities Matter in Angular

Because Angular templates also use:

```html id="entity3"
{{ value < 10 }}
```

So understanding escaping is important.

---

# 6. Meta Tags (VERY IMPORTANT)

Meta tags go inside `<head>`.

They help with:

* SEO
* Responsive design
* Browser behavior

---

## Character Encoding

```html id="meta1"
<meta charset="UTF-8">
```

Supports all languages.

---

## Responsive Design (IMPORTANT for Angular apps)

```html id="meta2"
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

Without this:

* Mobile apps break layout

---

## Description (SEO)

```html id="meta3"
<meta name="description" content="Learn Angular from scratch">
```

Used by Google search.

---

## Keywords (older SEO method)

```html id="meta4"
<meta name="keywords" content="Angular, HTML, CSS">
```

Less important today but still useful.

---

# 7. Complete Example Page (Media + Meta)

```html id="full1"
<!DOCTYPE html>
<html>

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Media demo page">

    <title>Media Page</title>
</head>

<body>

<h1>HTML Media Demo</h1>

<h2>Image</h2>
<img src="cat.jpg" alt="Cat Image" width="300">

<h2>Audio</h2>
<audio controls>
    <source src="song.mp3" type="audio/mpeg">
</audio>

<h2>Video</h2>
<video width="400" controls>
    <source src="video.mp4" type="video/mp4">
</video>

<h2>Iframe</h2>
<iframe
    width="400"
    height="250"
    src="https://example.com">
</iframe>

<p>5 &lt; 10 example using entities</p>

</body>

</html>
```

---

# Why This Matters for Angular

In Angular you will later use:

* `[src]` binding for images
* dynamic videos from APIs
* embedded components via `<iframe>`
* safe HTML rendering
* meta tags in `index.html`

Example Angular-style image:

```html id="ngimg"
<img [src]="user.imageUrl">
```
