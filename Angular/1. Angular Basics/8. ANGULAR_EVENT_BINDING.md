# Event Binding in Angular

## Introduction

Event Binding is an important concept in Angular.

Whenever you want to perform any action on the UI, Event Binding is used.

### Common UI Events:
- Mouse events
- Keyboard events
- Click events
- Change events
- Input events
- Select events

👉 All these come under **Event Binding**

This topic is usually covered in multiple parts because Angular supports many types of events.

---

## What is Event Binding?

### Simple Definition:
Event Binding is when:

- An event is declared in the HTML file (like click or input)
- A function written in the TypeScript file is called in response

👉 It connects **HTML events → TypeScript logic**

---

## First Example – Click Event

### HTML

```html id="btn1"
<button (click)="callMe()">Click Me</button>
````

### TypeScript (app.component.ts)

```ts id="ts1"
callMe() {
  alert('Hello Angular 21');
}
```

---

## Explanation

* `(click)` → event binding syntax
* `callMe()` → function in TypeScript file
* Parentheses `()` are required

👉 When the button is clicked, an alert appears:
**Hello Angular 21**

---

## Other Common Events

* `(change)` → used for dropdown or input changes
* `(input)` → used when typing in input fields

👉 These will be covered in detail later.

## Counter Example Using Event Binding

### Step 1: Declare Property

```ts id="counter1"
count = 0;
```

---

### Step 2: Create Function

```ts id="counter2"
counter() {
  this.count++;
  console.log(this.count);
}
```

---

### Step 3: Button Click

```html id="counter3"
<button (click)="counter()">+</button>
```

---

### Step 4: Display Value Using Interpolation

```html id="counter4"
<span>{{ count }}</span>
```

---

## Important Rule (Common Mistake)

* Inside **TypeScript file** → use `this.count`
* Inside **HTML file** → use `count` (no `this`)

---

## Counter with Plus and Minus Buttons

### HTML

```html id="counter5"
<button (click)="counter('plus')">+</button>
<button (click)="counter('minus')">-</button>

<span>{{ count }}</span>
```

---

### TypeScript

```ts id="counter6"
count = 0;

counter(action: string) {
  if (action === 'minus') {
    this.count--;
  } else {
    this.count++;
  }
}
```

---

## Prevent Counter from Going Below Zero

### Updated Logic

```ts id="counter7"
counter(action: string) {
  if (action === 'minus' && this.count > 0) {
    this.count--;
  } else if (action === 'plus') {
    this.count++;
  }
}
```

---

### Result

* `+` increases count
* `-` decreases count
* Counter never goes below 0

---

## Calling One Function from Another Function

### Example Function

```ts id="fn1"
showUserName() {
  alert('Hello');
}
```

---

### Calling from Button Click

```html id="fn2"
<button (click)="showUserName()">Show</button>
```

---

### Calling Inside Another Function

```ts id="fn3"
counter(action: string) {
  this.showUserName();
}
```

---

## Key Point

👉 Use `this.functionName()` to call another function inside TypeScript.
