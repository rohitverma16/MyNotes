# Data Types in Angular (TypeScript)

## What is a Data Type?

A data type defines what kind of value a variable, property, function, or event can hold.

Since Angular uses **TypeScript**, data types are very important for type safety and error prevention.

---

## Data Type for Property (Class Property)

```ts id="dt1"
data: number | string | boolean | undefined;
````

### Explanation:

* `data` is a **class property**
* It can hold multiple types using `|`
* This is called a **union type**

### Why use it?

* Prevents incorrect value assignment
* Provides flexibility when data type is not fixed

---

## Auto Type Detection

TypeScript can automatically detect types when a value is assigned:

```ts id="dt2"
data = 10;
```

### But:

* Manual typing is required when multiple types are possible
* Helps in maintaining strict control over data

---

## Function with Parameter Data Types

```ts id="dt3"
updateData(val: number, user: string) {
  this.data = val;
  console.log(user);
  console.log(this.sum(10, 20));
}
```

### Explanation:

* `val: number` → must be a number
* `user: string` → must be a string

👉 TypeScript ensures correct types at compile time

---

## Function Return Type

```ts id="dt4"
sum(a: number, b: number): number {
  return a + b;
}
```

### Explanation:

* Function returns a **number**
* Return type is defined after parameters using `: number`

---

## Calling Function with Parameters (HTML)

```html id="dt5"
<button (click)="updateData(60,'anil sidhu')">Update Data</button>
```

### Key Points:

* Passing a number and a string
* TypeScript checks types before execution
* Prevents runtime errors

---

## Event Handling with Data Types

```ts id="dt6"
handleEvent(event: PointerEvent | Event | MouseEvent) {
  console.log(event);
}
```

### Explanation:

* One function handles multiple event types
* Uses union type (`|`) for flexibility

---

## Event Binding Examples (HTML)

```html id="dt7"
<button (click)="handleEvent($event)">Click</button>

<input type="text" (change)="handleEvent($event)">

<div (mouseenter)="handleEvent($event)">
  <h1>Test event</h1>
</div>
```

### Event Types:

* `click` → PointerEvent
* `change` → Event
* `mouseenter` → MouseEvent

---

## Property vs Variable

### Property:

* Declared inside a class
* Accessible using `this`

```ts
data: number;
```

---

### Variable:

* Declared inside a function
* Uses `let` or `const`

```ts
let value = 10;
```

* Prevents wrong value assignment
* Helps other developers understand code clearly
* Catches errors before runtime (compile-time safety)
