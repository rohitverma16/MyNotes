# Build Your First Angular App (Hello World)

## Focus
Build your first Angular app and understand how the **“Hello World”** flow works in Angular.

---

## Creating the Angular Project

Start by opening the **Command Prompt / Terminal** and using Angular CLI to create a new project:

```bash
ng new hello-angular
````

During setup:

* Accept default options OR
* Choose CSS / SCSS based on preference

---

## Running the Angular App

Move into the project folder:

```bash
cd hello-angular
```

Start the development server:

```bash
ng serve --open
```

Now the browser opens automatically at:

```
http://localhost:4200
```

You will see the default Angular welcome page.

---

## Understanding Angular App Structure

Key folders and files:

* **src/** → Main application source code
* **app/** → Contains components and core app logic
* **main.ts** → Entry point of the application
* **index.html** → Base HTML file loaded in the browser

These form the basic structure of every Angular application.

---

## Modifying the “Hello World” Message

Open:

```
src/app/app.component.html
```

Replace the default content with:

```html
<h1>Hello Angular World!</h1>
```

Save the file.

The browser updates automatically due to Angular’s live reload feature.

---

## Understanding Components

A component is the core building block of Angular applications.

Each component consists of:

* HTML Template (UI)
* CSS Styles (Design)
* TypeScript Logic (Functionality)

The file **app.component.ts** connects all these parts together.

---

## Changing the Title from TypeScript File

Inside:

```
app.component.ts
```

Modify the class as follows:

```ts
export class AppComponent {
  title = 'Hello Angular World!';
}
```

---

## Using Interpolation in HTML

Display the TypeScript value inside HTML using interpolation:

```html
<h2>{{ title }}</h2>
```

Now the browser will show dynamic content based on the TypeScript variable.

---

## Auto-Reload Feature

Angular CLI provides **Hot Module Reloading (HMR)** by default.

This means:

* Any change in code automatically updates in the browser
* No manual refresh is needed
* Development becomes faster and smoother
