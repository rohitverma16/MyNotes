# Lesson 6: JavaScript Basics (Core Foundation for Angular)

## What is JavaScript?

JavaScript is a programming language used to:

* Make webpages interactive
* Handle user actions (clicks, inputs)
* Fetch data from APIs
* Control application logic

👉 Angular is built using JavaScript + TypeScript.

---

# 1. How JavaScript Runs

You can write JS in 3 ways:

## 1. Inside HTML

```html id="js1"
<script>
  alert("Hello");
</script>
```

---

## 2. External File (BEST)

```html id="js2"
<script src="app.js"></script>
```

---

## 3. Browser Console

Open:

* Right click → Inspect → Console

---

# 2. Variables (VERY IMPORTANT)

Variables store data.

## var (old)

```javascript id="js3"
var name = "John";
```

## let (modern)

```javascript id="js4"
let age = 25;
```

## const (fixed value)

```javascript id="js5"
const country = "India";
```

---

## Difference

| Type  | Can change? | Scope    |
| ----- | ----------- | -------- |
| var   | Yes         | Function |
| let   | Yes         | Block    |
| const | No          | Block    |

---

# 3. Data Types

```javascript id="js6"
let name = "John";     // string
let age = 25;          // number
let isActive = true;   // boolean
let user = null;       // null
let value;             // undefined
```

---

# 4. Operators

## Arithmetic

```javascript id="js7"
let sum = 10 + 5;
let diff = 10 - 5;
let mul = 10 * 5;
let div = 10 / 5;
```

---

## Comparison

```javascript id="js8"
10 > 5   // true
10 < 5   // false
10 == 10 // true
```

---

## Strict Comparison (IMPORTANT)

```javascript id="js9"
10 === "10" // false
```

👉 checks value + type

---

# 5. Functions

Functions are reusable blocks of code.

## Basic Function

```javascript id="js10"
function greet() {
  console.log("Hello");
}
```

Call it:

```javascript id="js11"
greet();
```

---

## Function with Parameter

```javascript id="js12"
function greet(name) {
  console.log("Hello " + name);
}

greet("John");
```

---

## Return Value

```javascript id="js13"
function add(a, b) {
  return a + b;
}

let result = add(5, 3);
```

---

# 6. Arrays (VERY IMPORTANT for Angular)

```javascript id="js14"
let users = ["John", "Mike", "Sara"];
```

---

## Access Items

```javascript id="js15"
console.log(users[0]); // John
```

---

## Array Methods

### push (add)

```javascript id="js16"
users.push("Alex");
```

### pop (remove last)

```javascript id="js17"
users.pop();
```

---

## Loop through array

```javascript id="js18"
for (let i = 0; i < users.length; i++) {
  console.log(users[i]);
}
```

---

# 7. Objects (VERY IMPORTANT)

Objects store key-value pairs.

```javascript id="js19"
let user = {
  name: "John",
  age: 25,
  city: "Delhi"

```

## Access values

```javascript id="js20"
console.log(user.name);
```

# 8. Arrays of Objects (Used in Angular a LOT)

# 9. ES6 Arrow Functions (VERY IMPORTANT)

```javascript id="js22"

  console.log("Hello");
};
```

Short form:

# 10. Conditions (if/else)

```javascript id="js24"

  console.log("Minor");
}
```

# 11. Loops

## for loop

```javascript id="js25"
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

---

## while loop

```javascript id="js26"
let i = 0;

while (i < 5) {
  console.log(i);
  i++;
}
```

---

# 12. Events (VERY IMPORTANT for Angular concept)

```html id="js27"
<button onclick="sayHello()">Click</button>

<script>
function sayHello() {
  alert("Hello!");
}
</script>
```

👉 Angular replaces this with `(click)`.

# 13. DOM Manipulation

## Change Content

# 15. Async / Await (VERY IMPORTANT)

# Why This is Important for Angular

In Angular you will use:

| JavaScript Concept | Angular Equivalent |
| ------------------ | ------------------ |
| onclick            | (click)            |
| DOM manipulation   | Data binding       |
| Functions          | Component methods  |
| Arrays             | *ngFor             |
| Objects            | Models             |
| Promises           | Observables (RxJS) |
| Async/Await        | HTTP calls         |
